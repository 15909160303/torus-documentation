---
title: Login with openlogin sdk
---

Call `openlogin.login` to trigger user login flow where it's appropriate in your
application lifecycle.

```js
    await sdkInstance.login({
      loginProvider: "google",
      redirectUrl: `${window.origin}/solana`,
    });
```

`openlogin.login` opens a login modal for your user to authenticate using
his/her prefer login method, including SSO, social logins, passswordless, and Touch/Face ID.
